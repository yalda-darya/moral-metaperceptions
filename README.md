# Moral Metaperceptions

This repository contains the code, materials, and outputs necessary to
reproduce the analyses reported in the paper:

Daryani, Y., & Dehghani, M. (in preparation).

## Repository Structure
- `data/`: processed datasets used for analyses
- `code/`: preprocessing, analysis, and visualization scripts
- `results/`: figures, tables, and model outputs
- `paper/`: manuscript and supplementary materials
- `environment/`: computational environment details

## Data Availability
Raw data cannot be shared due to IRB restrictions.
All analyses rely on anonymized, processed data or summary statistics.

## Reproducibility
Scripts are numbered and should be run in order.
All paths are relative.
